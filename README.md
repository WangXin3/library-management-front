# library-management-front
# 图书管理系统
技术栈：
* Vue2.0
* Element-UI
* 基于PanJiaChen大佬的vue-admin-template，地址：https://github.com/PanJiaChen/vue-admin-template
* ...

运行前置条件：NodeJs

运行步骤：
1. npm install
2. npm run dev
3. 用浏览器打开http://localhost:9528

对应后端项目地址：https://github.com/WangXin3/library-management-
